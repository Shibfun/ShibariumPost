Shibedia Usage Guide

Welcome to Shibedia, the fun way to create, explore, and mint MemeCoins on Shibarium! This guide shows you how to launch a MemeCoin, add details, browse tokens, mint them, and explains how tokens are shared.

Getting Started

Shibedia is an Instagram-like platform to view MemeCoins, search for tokens, or mint new ones. You don’t need a wallet to browse, but you’ll need one to create or mint.

What You Need

  • Web Browser: Chrome, Firefox, or any modern browser.
  • MetaMask: Install MetaMask for creating or minting MemeCoins.
  • BONE: Shibarium’s currency for transaction fees.

How to Use Shibedia

1. Create a MemeCoin

Launch your own MemeCoin on Shibarium.

  1. Go to https://shibfun.github.io/ShibariumPost/.
  2. Click Connect Wallet to link MetaMask to Shibarium.
  3. Enter a name (e.g., “Paw”) and symbol (e.g., “PAW”).
  4. Click “Deploy,” confirm in MetaMask (needs BONE), and note your token’s contract address.

2. Add Token Details

Add a picture and links to make your MemeCoin stand out.

  1. Visit https://shibfun.github.io/ShibariumPost/SetInfo/.
  2. Connect MetaMask (must be the token’s creator).
  3. Enter your MemeCoin’s contract address and click “Verify Owner.”
  4. Add:
    • Image: Square image URL (e.g., https://woofswap.finance/image/tokens/WOOF.png).
    • Description: Short text (e.g., “Paw is a meme coin on Shibarium.”).
    • Website: Your site (e.g., https://www.woofswap.finance).
    • Twitter: Twitter link (e.g., https://x.com/woofswap).
    • Telegram: Group link (e.g., https://t.me/ShibariumWeb3).
  5. Click “Set Metadata,” confirm in MetaMask, and see your token on Shibedia.

3. Browse MemeCoins

Explore MemeCoins on Shibedia.

  1. Open https://shibfun.github.io/ShibariumPost/Shibedia/.
  2. Browse:
    • View up to 100 recent MemeCoins in a grid, each with an image, name, description, and links.
    • No wallet needed.
  3. Search:
    • Enter a MemeCoin’s contract address in the search bar.
    • Click Search to see that token’s details.
    • Wrong address? You’ll see “Invalid contract address!”

4. Mint MemeCoins

Mint tokens to get some for yourself.

  1. Go to Shibedia.
  2. Click Connect Wallet to connect MetaMask.
  3. Choose a MemeCoin (from the grid or search).
  4. Click its Mint button (enabled after connecting).
  5. Confirm in MetaMask (needs BONE).
  6. Results:
    • Success: “Minted successfully!” You get 1 trillion tokens, and 200 random addresses get 5 billion each.
    • Failure: Error like “Total supply exceeded” if no tokens are left.

How MemeCoin Tokens Are Shared

Each MemeCoin has 1,000,000,000,000,000 * 10^18 (1 quintillion) tokens, split as follows:

  • Creator’s Share: The creator gets 100 trillion tokens (10%) when the MemeCoin is created.
  • Public Minting: Anyone can mint, receiving 1 billion tokens per mint, until no tokens remain.
  • Airdrop: Every mint sends 5 million tokens to each of 200 random addresses (1 billion total per mint).
    • Addresses: Up to 200 unique, randomly chosen addresses.
    • Tokens per Address: 5 million tokens each.
  • Total Available: After the creator’s 100 trillion, 900 trillion tokens are for minting and airdrops. Each mint uses 2 billion (1 billion for the minter, 1 billion for airdrops).
  • Rules:
    • Anyone with a wallet can mint, using BONE for fees.
    • Only real users (not smart contracts) can mint.
    • Minting stops when the total supply reaches 1 quintillion.

Example: For a MemeCoin like “Paw”:

  • Creator gets 100 trillion tokens at creation.
  • Each mint gives minter 1 billion tokens and airdrops 5 million to 200 random addresses.
  • Minting continues until 900 trillion tokens are used up.

Troubleshooting

  • Can’t Connect Wallet: Install or update MetaMask, add Shibarium, and retry.
  • Mint Button Disabled: Connect your wallet. If still grayed out, refresh the page.
  • Minting Fails:
    • Check BONE balance for fees.
    • If “Total supply exceeded,” the MemeCoin is fully minted.
    • Check network at ShibariumScan.
  • No Tokens Show: Refresh or check your internet.
  • Search Fails: Ensure the contract address is correct.
