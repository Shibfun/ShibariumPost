Shibedia Usage Guide

Welcome to Shibedia, your fun and easy way to explore, create, and mint MemeCoins on the Shibarium blockchain! This guide shows you how to deploy a MemeCoin, set its details, view tokens on Shibedia, and mint new tokens.

Getting Started

Shibedia is a web platform with an Instagram-like look, where you can browse MemeCoins, search for specific tokens, or mint new ones. You don’t need a wallet to view or search, but you’ll need one to create or mint tokens.

What You Need

  • Web Browser: Chrome, Firefox, or any modern browser.
  • MetaMask: Install the MetaMask extension for creating or minting MemeCoins.
  • BONE Tokens: Shibarium’s currency for transaction fees, needed for deploying, setting details, or minting.

How to Use Shibedia

1. Deploy a MemeCoin

Create your own MemeCoin on Shibarium using the deployment page.

  1. Visit https://shibfun.github.io/ShibariumPost/.
  2. Click “Connect Wallet” to link your MetaMask wallet to Shibarium.
  3. Enter your token’s name (e.g., “Paw”), symbol (e.g., “PAW”), and other details.
  4. Click “Deploy” and confirm the transaction in MetaMask (requires BONE for fees).
  5. Once confirmed, you’ll get a contract address for your new MemeCoin.

2. Set MemeCoin Details

Add a picture, description, and links to make your MemeCoin stand out.

  1. Go to https://shibfun.github.io/ShibariumPost/SetInfo/.
  2. Connect your MetaMask wallet (must be the token’s owner).
  3. Enter your MemeCoin’s contract address.
  4. Click “Verify Owner” to confirm you own the token.
  5. Fill in:
    • Image URL: Link to a square image (e.g., https://woofswap.finance/image/tokens/WOOF.png).
    • Description: Short text (e.g., “Paw is a meme coin on Shibarium.”).
    • Website: Your site (e.g., https://www.woofswap.finance).
    • Twitter: Twitter link (e.g., https://x.com/woofswap).
    • Telegram: Telegram group (e.g., https://t.me/ShibariumWeb3).
  6. Click “Set Metadata” and confirm in MetaMask.
  7. Your token’s details will appear on Shibedia.

3. View MemeCoins on Shibedia

Browse or search MemeCoins on the Shibedia page.

  1. Visit https://shibfun.github.io/ShibariumPost/Shibedia/.
  2. Browse:
    • The page shows up to 100 recent MemeCoins in a grid, like Instagram posts.
    • Each card has a square image, name (e.g., “Paw”), symbol (e.g., “PAW”), description, and links (website, Twitter, Telegram).
    • No wallet is needed to view.
  3. Search:
    • Enter a MemeCoin contract address in the search bar.
    • Click Search to show that token’s card.
    • If the address is wrong, you’ll see an error like “Invalid contract address!”

4. Mint MemeCoins

Mint new tokens from any MemeCoin to get some for yourself.

  1. Go to Shibedia.
  2. Click Connect Wallet to link MetaMask to Shibarium.
  3. Find a MemeCoin (in the grid or via search).
  4. Click the Mint button on its card (enabled after wallet connection).
  5. Confirm the transaction in MetaMask (needs BONE for fees).
  6. Results:
    • Success: You’ll see “Minted successfully!” and receive tokens.
    • Failure: An error shows, e.g., “Failed to mint: Total supply exceeded” if no more tokens can be minted.

MemeCoin Economics

Every MemeCoin on Shibedia follows simple rules for how tokens are created and distributed:

  • Total Tokens: Each MemeCoin has a maximum of 1 quintillion tokens (1,000,000,000,000,000 * 10^18).
  • Pre-Minted: 10% of tokens (100 trillion) are automatically given to the creator (owner) when the MemeCoin is deployed.
  • Public Minting: The remaining 90% (900 trillion) can be minted by anyone.
    • Each mint creates a fixed amount (usually 100 trillion tokens).
    • Anyone with a wallet can mint, as long as there are tokens left.
    • The contract stops minting when the total supply hits the maximum.
  • Rules:
    • Only real users (not smart contracts) can mint.
    • You need BONE to pay transaction fees.
    • Minting stops automatically when all tokens are created.

Example: For a MemeCoin like “Paw”:

  • Owner gets 100 trillion tokens at creation.
  • Anyone can mint 100 trillion tokens per transaction, up to 900 trillion total.
  • Once 1 quintillion tokens are minted, no more can be created.

Troubleshooting

  • Can’t Connect Wallet: Install or update MetaMask, ensure Shibarium is added, and try again.
  • Mint Button Disabled: Connect your wallet. If still grayed out, refresh the page.
  • Minting Fails:
    • Ensure you have enough BONE for fees.
    • If “Total supply exceeded,” the MemeCoin is fully minted.
    • Check network status at ShibariumScan.
  • No Tokens Show: Refresh the page or check your internet connection.
  • Search Fails: Verify the contract address is correct.
