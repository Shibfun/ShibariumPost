Shibedia Usage Guide

Welcome to Shibedia, a vibrant platform to explore and interact with MemeCoins on the Shibarium blockchain. This guide walks you through how to use the Shibedia page to view, search, and mint MemeCoins, and explains the MemeCoin format displayed on the site.

Overview

Shibedia is a web-based interface styled like Instagram, showcasing MemeCoins created via the ShibariumMemeCoin contract (address: 0x9491FD7AB3651904F2D5282bFa97f052698ff8b3) on Shibarium (chain ID: 109). It features a waterfall (masonry) layout of up to 100 recent MemeCoins, a search function for specific tokens, and a minting feature for users with a connected wallet.

  • View MemeCoins: Browse the latest 100 MemeCoins without a wallet.
  • Search Tokens: Find a specific MemeCoin by its contract address.
  • Mint Tokens: Mint new tokens from a MemeCoin contract after connecting a wallet.

Getting Started

Requirements

  • Web Browser: Chrome, Firefox, or any modern browser.
  • MetaMask (Optional): Install the MetaMask extension for minting. Not required for viewing or searching.
  • BONE Tokens: Shibarium’s native token for gas fees, needed for minting.

Using Shibedia

1. Viewing MemeCoins (Waterfall Layout)

Upon loading, Shibedia displays a waterfall grid of up to 100 latest MemeCoins, fetched from the ShibariumMemeCoin contract’s getLatestMemeCoins function. No wallet connection is needed.

  • Display: Each MemeCoin appears as a card with:
    • Header: Token icon (from metadata Image URL or placeholder) and name with symbol (e.g., “Paw (PAW)”).
    • Image: Square image (1:1, from metadata Image URL, e.g., https://woofswap.finance/image/tokens/WOOF.png).
    • Content: Token name, description (e.g., “Paw is a meme coin on Shibarium.”), and links (Website, Twitter, Telegram).
    • Mint Button: Labeled “Mint,” disabled (grayed out) until a wallet is connected.
  • Behavior: Cards load with a fade-in animation, arranged in a responsive masonry layout (multi-column on desktop, single-column on mobile).
  • Fallbacks:
    • No tokens: Shows “No MemeCoins found.”
    • Missing metadata: Uses placeholders (e.g., https://via.placeholder.com for images, “No description” for text).

2. Searching for a MemeCoin

You can search for a specific MemeCoin by its contract address without a wallet.

  1. Locate the search bar at the top, labeled “Search Meme Coin by Contract Address.”
  2. Enter a valid MemeCoin contract address (e.g., a token created by 0x9491FD7AB365...).
  3. Click the Search button (orange-red gradient).
  4. Result:
    • The waterfall clears, showing a single card with the token’s details (same format as above).
    • If the address is invalid or not a MemeCoin, an error appears: “Invalid contract address!” or “Failed to fetch token.”

Note: To return to the full waterfall, refresh the page or clear the search input and search again.

3. Minting MemeCoins

Minting creates new tokens for a MemeCoin, requiring a MetaMask wallet connected to Shibarium with BONE for gas fees.

  1. Connect Wallet:
    • Click the Connect Wallet button (top, orange-red gradient).
    • MetaMask opens, prompting you to connect your account.
    • If not on Shibarium (chain ID: 109), MetaMask will switch or add the network (RPC: https://www.shibrpc.com, native token: BONE).
    • Success: Button updates to “Connected: 0x123...456” (shortened address), and a “Connected to Shibarium” message appears.
  2. Mint a Token:
    • After connecting, all Mint buttons in the waterfall or search results become enabled (no longer grayed out).
    • Find a MemeCoin card (in the waterfall or via search) and click its Mint button.
    • MetaMask opens to confirm the transaction (gas limit: 3,000,000, gas price: 1.5x network average).
    • Outcomes:
      • Success: “Minted successfully!” appears, and the button remains enabled for further mints (if allowed by the contract).
      • Failure: Error message shows, e.g., “Failed to mint: Total supply exceeded” (if the contract’s totalSupply + MINT_AMOUNT > MAX_SUPPLY) or network issues.

Note: The MemeCoin contract allows anyone to mint (as long as msg.sender == tx.origin and totalSupply + MINT_AMOUNT <= MAX_SUPPLY). If minting fails, check BONE balance or contract state via ShibariumScan.

MemeCoin Format

Each MemeCoin on Shibedia is a token created by the ShibariumMemeCoin contract, with a standardized format displayed in the waterfall and search results. Below is the structure and how it’s presented:

Contract Details

  • Contract: MemeCoin (deployed by 0x9491FD7AB365...).
  • Network: Shibarium (chain ID: 109).
  • Max Supply: 1,000,000,000,000,000 * 10^18 (1 quintillion tokens).
  • Mint Amount: Fixed amount per mint (defined in the contract, typically 100 trillion tokens).
  • Functions Used:
    • name(): Returns the token’s name (e.g., “Paw”).
    • symbol(): Returns the token’s symbol (e.g., “PAW”).
    • getMetadata(): Returns [Image URL, Description, Website, Twitter, Telegram].
    • mint(): Mints new tokens, restricted by supply and EOA check.

Displayed Format

Each MemeCoin card follows this structure:

  • Header:
    • Icon: 40x40px circular image from metadata Image URL (e.g., https://woofswap.finance/image/tokens/WOOF.png) or placeholder (https://via.placeholder.com/40).
    • Title: “Name (Symbol)” (e.g., “Paw (PAW)”). Fallback: “Unknown (Unknown)”.
  • Main Image:
    • Square (1:1 aspect ratio), full card width (min 280px), from metadata Image URL or placeholder (https://via.placeholder.com/280).
    • Example: https://woofswap.finance/image/tokens/WOOF.png.
  • Content:
    • Name: Bold title (e.g., “Paw”).
    • Description: Short text from metadata (e.g., “Paw is a meme coin on Shibarium.”), truncated to ~60px height with ellipsis. Fallback: “No description.”
    • Links:
      • Website: Displays hostname (e.g., “woofswap.finance”) from metadata, linked to full URL (e.g., https://www.woofswap.finance). Fallback: “No website” (unlinked).
      • Twitter: Displays handle (e.g., “@woofswap”) from metadata, linked to URL (e.g., https://x.com/woofswap). Fallback: “No Twitter”.
      • Telegram: Displays group name (e.g., “ShibariumWeb3”) from metadata, linked to URL (e.g., https://t.me/ShibariumWeb3). Fallback: “No Telegram”.
    • Each link has a 16x16px icon (website, Twitter, Telegram) for visual appeal.
  • Mint Button:
    • Labeled “Mint,” orange-red gradient, centered.
    • Disabled (gray) without a wallet; enabled for any connected wallet.
    • Triggers mint(), with contract enforcing supply limits.

Troubleshooting

  • Mint Button Grayed Out: Ensure wallet is connected. If still disabled, refresh the page and reconnect.
  • Mint Fails:
    • Check BONE balance for gas fees.
    • If “Total supply exceeded,” the token has reached its max supply.
    • If network error, verify Shibarium RPC (https://www.shibrpc.com) or try https://rpc.shibariumchain.com.
  • No Tokens Load: Check network connection or RPC status. Refresh the page.
  • Invalid Address: Ensure the entered address is a valid MemeCoin contract.
  • MetaMask Issues: Update MetaMask, ensure Shibarium is added, and retry.
